# Project Starlit

**Project Starlit** is a web application built using **Firebase Studio**. It integrates frontend, backend, and Firebase services for a full-stack solution.

## Features
- Responsive web application
- Firebase backend for authentication, database, and hosting
- Modular UI components for easy maintenance
- Real-time updates and interactions

## Project Structure
```

├── src/
│   ├── app/           # Pages and global styles
│   ├── components/    # Reusable UI components
│   └── ai/            # AI utilities
├── package.json
├── tailwind.config.ts
├── tsconfig.json
├── firebase.json
└── README.md

````

## Installation
1. Clone the repository:
```bash
git clone https://github.com/awais383/ProjectStarlit.git
````

2. Go to the project directory:

```bash
cd ProjectStarlit
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

## Deployment

Deploy the project to Firebase Hosting:

```bash
firebase deploy
```
